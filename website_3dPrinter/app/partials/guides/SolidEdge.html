Solid Edge

Solid Edge ist ein CAD-Programm. Für Studenten und andere an der Ausbildung Beteiligte ist eine kostenlose akademische Lizenz erältlich. Folgend wird die grundlegende Bedienung vorgestellt.

Parallel oder Sequenziell?

Die programminterne Verarbeitung wurde vor wenigen Jahren überarbeitet. Vorher wurde der gesamte Verlauf des Designs gespeichert. Vorteil waren eindeutige Abhängigkeiten zwischen Bauteilmaßen. Da bei Änderungen allerdings der gesamte Stack neu berechnet werden musste, wurde das Programm mit steigender Objektgröße immer unperformanter. Daher wurde die parallele Verarbeitung eingeführt. Hier muss das Design weniger strikten Schritten folgen und die Entwicklungen können schneller angepasst werden.

Schritt 0: Neues Bauteil anlegen

Ein Bauteil wird mit *.par bezeichnet. Mehrere Bauteile können später zu einer Assembly zusammengefasst werden.

Schritt 1: Grundfläche

Im ersten Schritt wird in eine Raumebene eine 2D-Grundfläche eingezeichnet. Mit dem Würfel unten rechts kann die Raumansicht schnell verändert werden. Die Bemaßungen der gezeichneten Fläche können direkt eingegeben werden. Auch auf bestehenden Flächen von Objekten können neue Flächen eingezeichnet werden.

Schritt 2: Extrusion

Als nächstes wird die Fläche zu einem Volumen extrahiert. Dies kann mit dem Button oben geschehen. Alternativ wird bei Auswählen einer Fläche ein senkrechter Pfeil zum Extrahieren eingeblendet. Eine negative Extrusion erzeugt eine Bohrung.

Schritt 3: Assembly

Aus verschiedenen Bauteilen lässt sich nun ein Gesamtobjekt erstellen. Hierfür wird eine neue *.asm-Datei angelegt und die gewünschten Bauteile aus der Teilebibliothek rechts im Fenster hinzugefügt. Durch Verschieben und die Option Komponentenmontage lassen sich die einzelnen Kompenenten zu einem Objekt kombinieren.

Schritt 4:Als STL speichern

Über das "Speichern unter"- Menue lässt sich das entworfene Objekt als übersetzt speichern. Zielformat für Cura ist STL. Hierbei muss im Speichern-Dialog über Optionen die Einheit auf Milimeter geändert werden, da sonst die Dateien in Cura als zu klein interpretiert werden.
