Meshmixer

Dieser Artikel bezieht sich auf die MeshMixer in der Version 10.9.246

Allgemein

Meshmixer ist eine kostenlose Software zum Bearbeiten von 3D-Modellen. Die Software hat einen großen Fundus an nützlichen Funktionen, besonders für den 3D-Druck. Im folgenden werden die wichtigsten Werkzeuge vorgestellt um ein Modell auf den Druck vorzubereiten.

Die wichtigsten Tools

Fehleranalyse

Ein 3D-Modell kann verschiedene Fehler haben, die unter Umständen optisch nicht sichtbar sind, aber dennoch zu Fehlern in den Algorithmen des Slicers führen können.
Viele diese Probleme lassen sich mit dem Inspector Werkzeug in MeshMixer überprüfen und teilweise automatisch lösen. Das Werkzeug erkennt die folgenden Probleme und markiert sie mit farbigen Kugeln:
  • Blau: Löcher in der Oberfläche des Objektes
    Löcher können entweder offensichtliche offene Stellen im Modell sein, oder entstehen, wenn zwei Kanten exakt übereinander liegen, aber nicht verbunden sind.
  • Rot: Nicht-Manigfaltiger Abschnitt
    Dies sind Teile des Objektes die so in der Realität nicht existieren könnten. z.B. Abschnitte ohne Dicke, Flächen im inneren eines Objektes, überlappende Flächen, etc. Siehe auch hier
  • Magenta: Kleine, nicht mit dem Hauptobjekt verbundene Teile
    Besonders beim Scannen mit der Kinect kann es vorkommen, das kleine Artefakte entstehen, die losgelöst vom Hauptobjekt und im Vergleich dazu sehr klein sind. Diese Objekte sind meistens nicht gewünscht, erhöhen die Druckzeit und sorgen im schlimmsten Fall für Probleme beim Druck.
Durch einen Klick auf "Auto Repair All" versucht MeshMixer alle gefundenen Probleme automatisch zu beheben. Sollte das nicht gewünscht sein, kann durch einen Linksklick auf eine der Kugeln eine Reperatur nur auf eine Problemstelle angewendet werden. Ein Rechtsklick auf eine Kugel seleketiert die problematische Stelle, sodass eine manuelle Lösung gefunden werden kann. Das Werkzeug befindet sich unter Analysis -> Inspector.

Überhanganalyse & Supports

Eines der wichtigsten Analysetools. Es untersucht das Modell auf Überhänge deren Neigung über einem bestimmten Winkel liegt, was im Druckprozess unschöne oder fehlerhafte Ergebnisse erzeugen können. Die in rot markierten Bereiche sind solche kritischen Stellen, an denen Stützen angebracht werden sollten um das Objekt zu tragen. Integriert in dieses Tool ist die Möglichkeit Stützen automatisch zu generieren.
WICHTIG: Bevor die Stützen generiert werden, muss das Modell auf die finale Größe skaliert werden!
Für den Ultimaker 2 ist ein Preset vorhanden, das im Dropdown Menü ausgewählt werden kann. Ein Klick auf [1] Generate Supports berechnet dann die notwendigen Stützen. Die automatisch generierten Supports können nun nachbearbeitet werden, bis ein außreichendes Gerüst vorhanden ist. Dabei gilt: Mehr Stützen liefern in der Regel bessere Ergebnisse, verlängern aber auch die Druckzeit. Anpassungen können auf folgende Arten vorgenommen werden:
  • Stützen entfernen
    Ungewünschte Stützen lassen sich durch einen Klick mit gedrückter STRG Taste entfernen.
  • Stützen automatisch hinzufügen
    Ein Klick auf eine Stütze oder einen rot markierten Überhang erzeugt automtisch eine neue Verstrebung zu diesem Punkt. Dabei versucht der Algorithmus intelligent möglichst kurze Stützen zu generieren.
  • Stützen manuell ziehen
    Durch ein Klicken und Ziehen auf rot markierte Überhänge oder andere Stützen, lässt sich eine neue Verbindung erzeugen.
Wenn ausreichend viele Stützen im Modell vorhanden sind, können diese durch einen Klick auf [2] Convert to Solid permanent zum Modell hinzugefügt werden.

Stabilitätsanalyse

Dieses Tool gibt schon vor dem Druck einen Eindruck von der Standfestigkeit des fertigen Objekts.
Das Stabilitätstool berechnet den Schwerpunkt des Modells und lässt somit Rückschlüsse zu, ob das gedruckte Objekt aufrecht stehen kann. Dabei werden jedoch Faktoren wie Infill nicht beachtet, die Angabe dient also nur als Orientierung. Das Werkzeug befindet sich unter Analysis -> Stability

Stärkeanalyse

Dieses Tool gibt schon vor dem Druck einen Eindruck von potenziellen Schwachstellen am fertigen Objekt.
Das Stärketool untersucht das Modell auf potenzielle Schwachstellen, also Stellen die am fertigen Objekt eventuell abbrechen könnten. Grün markierte Bereiche sind unproblematisch, gelbe oder sogar rote Farben geben eine Schwachstelle an. Auch hier gilt: Es wird von einem soliden Objekt ausgegangen, etwaige Füllmuster können das Ergebnis beeinflussen. Das Werkzeug befindet sich unter Analysis -> Strength

Export

Um das Modell als STL Datei zu exportieren muss der Punkt File -> Export gewählt werden. Als Dateityp wird STL Binary Format (*.stl) ausgewählt.
